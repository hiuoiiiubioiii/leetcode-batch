# Pattern Extraction Framework

## Layer 1 — Raw Signals

From each problem row:
- problem statement features
- constraints
- input/output structure
- tags (if present)

## Layer 2 — Derived Signals

- data structure used
- traversal style
- optimization technique
- recurrence pattern

## Layer 3 — Core Patterns

Examples:
- two pointer
- sliding window
- binary search
- graph traversal
- dp state transition

## Layer 4 — Composite Patterns

Examples:
- sliding window + hashmap
- dfs + backtracking
- dp + bitmask

## Layer 5 — Meta Buckets

- search
- optimization
- counting
- partitioning
- greedy

## Goal

Turn every question into:

Question → Signals → Pattern(s) → Composite → Bucket → Strategy Template
